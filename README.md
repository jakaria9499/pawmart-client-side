# PawMart Client

Welcome to **PawMart Client**, a modern React-based e-commerce platform for pet products. This project allows users to browse, search, and Adoption pets, shop for pet food, accessories, and care products seamlessly.

## Features

- **User-friendly Interface:** Clean and intuitive UI for easy navigation to shopping and Adoption.  
- **Responsive Design:** Fully responsive layout that works on desktop, tablet, and mobile devices.  
- **Product Categories:** Browse products by categories such as Pets, Food, Accessories, and Care Products.  
- **Filter:** Quickly find products category filters.  
- **Secure Authentication:** Login and register users securely with Firebase Authentication.  
- **Shopping Cart:** Add, remove, and update products  in real-time.
